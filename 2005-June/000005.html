Hola, siguiendo el tutorial de "corriendo Luca en distros modernas" llego a 
hacer todo hasta el chroot. El problema viene cuando ejecuto el script me 
sale lo siguiente:

access control disabled, clients can connect from any host

Bienvenido a Luca v0.2.0

GTK-Warning **: cannot open display: :0.0


¿Qué puede ser?. Aclaro que tuve que alterar algunos pasos en el tutorial 
para lograr llegar hasta este último punto.

Alteré lo siguiente:

1) En el punto " y la populamos" la línea 2: $ ./sql/admin_period.py papo 
papo locahost" tuve que hacerlo después de hacer chroot y funcionó. ¿Será 
por la versión de Python que tengo instalada?

2) Tuve que alterar el script de ejecución: donde dice "sudo chroot 
/var/lib....bla, bla", puse: "sudo /usr/sbin/chroot ....bla, bla" . Si no 
saltaba un error de que no encontraba el comando chroot cuando ejeutaba el 
script como usuario no root. También tuve que alterar el sudoers donde el 
artículo decía que había que poner: " ALL= NOPASSWD: /bin/chroot" yo puse 
ALL= NOPASSWD: /usr/sbin/chroot"

PD: Uso Mandrake 10.0
