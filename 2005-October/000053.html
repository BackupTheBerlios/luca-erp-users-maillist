Hola, estoy teniendo las siguientes dudas con Luca v0.2.0 (uso el luca
enjaulado):

1) El sistema instalado está instalado en una Librería, dicha librería
trabaja con libros en consignación. El asunto es que necesitamos
liquidar ventas, me parece que el Luca no trae esta funcionalidad,
pero ¿existe alguna forma alternativa para alivianar el trabajo de
hacerlo manualmente?.
2) ¿Cómo hago para configurar una impresora en el Luca? ¿Debo
configurarla en el woody enjaulado?¿Recomiendan algún how-to?
3)Hemos tenido problemas para dar de alta un proveedor, más
específicamente con un CUIT de dicho proveedor. Cuando intentámos
guardar el sistema nos dice que no se pueden insertar claves
duplicadas (obviamente, ese cuit no estaba registrado previamente en
el sistema). Si cambiamos el cuit, funciona perfectamente.
4) Algo muy importante que no podemos hacer es el ajuste de stock
inicial. Para hacerlo debemos crear un remito e indicar las
cantidades, luego podemos hacer todas las modificaciones que
querramos. ¿Es así como debe hacerse?.
5) ¿Se puede imprimir facturas con el Luca? o ¿necesito hacer facturas
pre-impresas?, disculpen la ignorancia, es la primera vez que trabajo
con un sistema de facturación.

Saludos a todos y muchísimas gracias. Ramiro Laso
